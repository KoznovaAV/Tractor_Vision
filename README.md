# Tractor Vision: Классификация моделей и состояния тракторов

## 📋 Описание проекта
Проект направлен на разработку системы компьютерного зрения для автоматического распознавания моделей тракторов (МТЗ, Кировец, ЧТЗ и др.) и определения их технического состояния (грязный/чистый).

##  Задачи
1. **Мультиклассовая классификация**: Определение конкретной модели трактора.
2. **Бинарная классификация**: Определение состояния (грязный / чистый).

## 🛠 Стек технологий
- **Язык**: Python 3.11
- **Deep Learning**: PyTorch, PyTorch Lightning
- **Архитектуры**: ConvNeXt, EfficientNet (через библиотеку `timm`)
- **Аугментации**: Albumentations
- **Оптимизация гиперпараметров**: Optuna
- **Мониторинг экспериментов**: Weights & Biases (W&B)
- **Анализ данных**: Pandas, Scikit-learn, Matplotlib, Seaborn

## 📁 Структура проекта
- `configs/` — YAML конфиги для экспериментов
- `data/` — датасеты (raw и processed)
- `notebooks/` — Jupyter ноутбуки для EDA (анализа данных)
- `src/` — исходный код проекта
  - `data/` — загрузка и аугментация данных
  - `models/` — архитектуры нейросетей
  - `training/` — логика обучения (PyTorch Lightning)
  - `utils/` — вспомогательные утилиты
- `weights/` — сохраненные чекпоинты моделей
- `output/` — результаты инференса и визуализации
